jQuery autoComplete
===================
This is a rewrite of `LakTEK <http://laktek.com>`_'s `smartAutoComplete <https://github.com/laktek/jQuery-Smart-Auto-Complete>`_ plugin.
::

    $('#example1').autoComplete({
        delay: 125,
        maxResultCount: 5,
        minCharacterCount: 1,
        separator: ',',
        sourcePacks: ['Avocado', 'Banana', 'Orange', 'Mango']
    }).focus();
